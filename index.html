Button Tags

Use the button classes on an <a> , <button> or <input> element.

Example
Link 

<a class="btn btn-primary btn-md btn-square" href="#" role="button">Link</a>

<button class="btn btn-primary btn-md btn-square" type="submit">Button<,/button>

<input class="btn btn-primary btn-md btn-square" type="button" value="Input">

<input class="btn btn-primary btn-md btn-square" type="submit" value="Submit">
Note:

Button shapes

Buttons shapes are available in three different shapes choose what suits to your project. .btn-square ,.btn-rounded , .btn-solid

Example
btn-square btn-solid btn-rounded 

<a class="btn btn-square" href="#" role="button">btn-square</a>

<a class="btn  btn-solid" href="#" role="button">btn-solid</a>

<a class="btn btn-rounded" href="#" role="button">btn-rounded</a>
