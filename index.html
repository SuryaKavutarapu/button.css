Button.css

A collection of buttons to be applied to links, buttons, input elements. Easily apply to your own projects, modify or just use for Design. Available in CSS file.

Download on Github

Button Tags

Use the button classes on an <a> , <button> or <input> element.

Example:-
Link 

<a class="btn btn-primary btn-md btn-square" href="#" role="button">Link</a>

<button class="btn btn-primary btn-md btn-square" type="submit">Button<,/button>

<input class="btn btn-primary btn-md btn-square" type="button" value="Input">

<input class="btn btn-primary btn-md btn-square" type="submit" value="Submit">
Note:

button varients

Buttons comes with two varients 1.outline varient .btn-default , 2. fill varient .btn-default-fill

Example:-
btn outline btn fill 

<a class="btn btn-primary" href="#" role="button">btn-outline</a>

<a class="btn btn-primary-fill" href="#" role="button">btn-fill</a>
Note:-

Outline button class don't need any addition to .btn-primary.If you want fill button you have to add .btn-primary-fill to get the fill effect.

Button shapes

Buttons shapes are available in three different shapes choose what suits to your project. .btn-square ,.btn-rounded , .btn-solid

Example:-
btn-square btn-solid btn-rounded 

<a class="btn btn-square" href="#" role="button">btn-square</a>

<a class="btn  btn-solid" href="#" role="button">btn-solid</a>

<a class="btn btn-rounded" href="#" role="button">btn-rounded</a>

button disabled

An special class also availble which disables button. .disabled

Example:-
btn disabled btn-normal 

<a class="btn btn-disturbed disabled" href="#" role="button">btn-disabled</a>
Note:-

This class uses pointer-events: none, to try to disable the link functionality of <a>s, but that CSS property is not yet standardized and isn't fully supported in Opera 18 and below, or in Internet Explorer 11. In addition, even in browsers that do support pointer-events: none, keyboard navigation remains unaffected, meaning that sighted keyboard users and users of assistive technologies will still be able to activate these links. So to be safe, use custom JavaScript to disable such links.
